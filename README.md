# digiital clock Synced with internet

# Features

* RTC with LSE
* GUI with STemWin
* Connected to WIFI through esp8266
* based on FreeRTOS
* Real Time meteo station
* indoor meteo using the IKS03 extension board

# Hardware

* STM32F769 discovery
